Titre : 💰 Budget Pro - Gestionnaire de Finances Personnel

Description : Budget Pro est une application mobile complète développée avec Flutter, conçue pour offrir une gestion financière intuitive, sécurisée et 100% hors-ligne. Elle permet aux utilisateurs de suivre leurs dépenses, gérer leurs dettes, planifier leur épargne et analyser leur santé financière.

Fonctionnalités Clés :

📊 Tableau de Bord Intelligent : Solde en temps réel, jauge de budget mensuel et graphiques interactifs.

📈 Analyses Visuelles :

Courbe d'évolution "Smart Chart" (Change de couleur Vert/Rouge selon le solde).

Diagramme circulaire pour la répartition des dépenses.

🤝 Gestion des Dettes : Suivi précis des créances ("On me doit") et dettes ("Je dois").

🐷 Objectifs d'Épargne : Cagnottes virtuelles avec barre de progression pour vos projets.

📅 Abonnements : Calcul automatique des coûts fixes mensuels et annuels.

🔒 Sécurité Maximale : Verrouillage biométrique (Empreinte/FaceID) et stockage local chiffré.

🛠 Outils Pro : Export PDF, Mode Sombre, Multi-devises dynamique, Recherche, Notifications de rappel.

Stack Technique :

Framework : Flutter & Dart

Base de Données : Hive (NoSQL local)

Packages : fl_chart, local_auth, pdf, flutter_local_notifications, provider.
